Privacy Policy for AI Scribe

Last updated: January 7, 2026

Introduction

AI Scribe is an audio and image transcription application. This privacy policy explains how we handle your data.

Data Collection and Processing

Audio Data

  • Local Processing (Gemma Mode): When you use Gemma 3n E2B local transcription, all audio and image processing happens entirely on your device. No audio data is transmitted to external servers.
  • Cloud Processing (Google Cloud API Mode): When you choose Google Cloud API for transcription, your audio and image data are sent to Google Cloud servers for processing.

Transcriptions

All transcriptions are stored locally on your device. We do not have access to your transcriptions.

Data We Do NOT Collect

We do not collect: personal information, usage analytics, location data, device identifiers, or any audio recordings.

Third-Party Services

When using Cloud API mode, audio data is processed by Google Cloud API, subject to Google Cloud Privacy Policy.

Data Security

All network communications use HTTPS encryption. No data is transmitted when using offline (Gemma Local) mode.
